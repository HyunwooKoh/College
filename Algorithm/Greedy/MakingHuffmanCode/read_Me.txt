This program is a program that receives a single English file, examines the frequency of each English character (including space characters), and uses it to generate a Huffman tree (decode tree).
each file means.

counter : read the file and count frequency of each character.
HeapNode.java : the node class for Minimum heap
Minheap : makes Minimum heap using the data from counter
TreeNode : the node class for huffman Tree
Huffman_Tree : which makes huffman Tree using Minimum heap
test : which is role of the main function.
